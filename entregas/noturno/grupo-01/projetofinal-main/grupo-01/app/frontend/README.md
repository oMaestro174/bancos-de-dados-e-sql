# Front-end — Consulta de Alunos

Aplicação React para consulta de alunos da Escola ITEAM RR.

## Tecnologias

- React
- CSS
- Vite

## Como executar

```bash
npm install
npm run dev
```

## Observação

Nesta primeira versão, os dados estão mockados em `src/data/mockData.js`.
Quando a API Node.js + Express + Prisma estiver pronta, substituir os dados mockados por chamadas HTTP para o back-end.
